# Test de Quotient Émotionnel (QE)

Ce projet permet d'évaluer votre **Quotient Émotionnel (QE)** via un questionnaire interactif en ligne. Le test calcule automatiquement vos scores pour 6 compétences émotionnelles et fournit une interprétation personnalisée de vos résultats.
